# SEOBoostPro

SEOBoostPro is a minimal starter for a full-stack SEO optimization platform.

## Getting Started

1. cd backend -> npm install -> npm run dev
2. cd frontend -> npm install -> npm run dev

### Environment Variables

Create a backend/.env file with the following keys:

- MONGODB_URI
- PAGESPEED_API_KEY
- LLM_API_KEY
- FRONTEND_BASE_URL
